ToyZilo • GitHub Pages Deploy
=================================
1) Create a new GitHub repo, e.g., toyzilo-prototype.
2) Upload these files at the repo root (index.html, styles.css, script.js, data.json, /assets/logo.png).
3) In GitHub, go to Settings → Pages → Build and deployment:
   - Source: Deploy from a branch
   - Branch: main (or master) / root
4) Wait ~1 min. Your site will be live at: https://<your-username>.github.io/toyzilo-prototype

Google Drive alternative
------------------------
• Upload the same folder to a static host or use Drive-to-web service that serves HTML with JS.
